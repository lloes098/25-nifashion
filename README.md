# 25-nifashion

Next.js 웹앱 프로젝트입니다.

## 시작하기

먼저 의존성을 설치하세요:

```bash
npm install
# 또는
yarn install
# 또는
pnpm install
```

개발 서버를 실행하세요:

```bash
npm run dev
# 또는
yarn dev
# 또는
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인하세요.

## 스크립트

- `npm run dev` - 개발 서버 실행
- `npm run build` - 프로덕션 빌드
- `npm run start` - 프로덕션 서버 실행
- `npm run lint` - ESLint 실행
